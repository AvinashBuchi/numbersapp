# Spring Boot Numbers App

A REST API built with Spring Boot that performs operations on a list of integers.

## Features
- Sum of positive numbers
- Count of positive numbers
- Maximum value
- Average calculation
- Input validation
- Global exception handling
- Unit tests (JUnit 5)
- Controller tests (MockMvc)
- Logging with SLF4J

## Tech Stack
- Java 17
- Spring Boot 4
- Maven
- Lombok
- JUnit 5
- Mockito
- MockMvc
